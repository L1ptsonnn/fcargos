# FCargos - Логістична платформа

Сучасна веб-платформа для логістичних перевезень, що з'єднує компанії та перевізників.

## 🚀 Особливості:

### Для компаній:
- 📍 **Створення маршрутів** з інтерактивною картою та автоматичним геокодуванням
- 💰 **Система ставок** - перевізники можуть пропонувати свої ціни
- 📊 **Відстеження вантажів** в реальному часі з прогрес-баром та картою
- 💬 **Чат з перевізниками** для обговорення деталей доставки
- ⭐ **Система рейтингів** - оцінюйте перевізників після виконання доставки
- 📈 **Статистика маршрутів** - аналітика по всіх ваших доставках
- 🗺️ **Історія маршрутів** з детальною інформацією про вагу, об'єм, перевізників

### Для перевізників:
- 🔍 **Пошук доступних маршрутів** з фільтрацією за містом
- 💵 **Створення ставок** через зручний модальний інтерфейс
- 📱 **Відстеження активних доставок** з оновленням прогресу
- ⭐ **Рейтингова система** - отримуйте оцінки від компаній
- 💬 **Прямий чат** з компаніями
- 📊 **Статистика доходів** та завершених маршрутів
- 👤 **Профіль з рейтингом** для привернення нових клієнтів

### Загальні функції:
- 🚚 **Динамічна карта світу** з візуалізацією всіх маршрутів у реальному часі
- 👥 **Двотипна реєстрація**: окремі форми для компаній та перевізників
- 🗺️ **Вибір адреси на карті** при реєстрації
- 🔔 **Система сповіщень** про нові ставки, повідомлення та просрочені маршрути
- 🎨 **Сучасний дизайн** з градієнтами, анімаціями та ефектами
- 📱 **Адаптивний інтерфейс** для всіх пристроїв

## 🔧 Встановлення

### Вимоги
- Python 3.8+
- pip
- Git

### Кроки встановлення

1. **Клонуйте репозиторій:**
```bash
git clone https://github.com/YOUR_USERNAME/fcargos.git
cd fcargos
```

2. **Створіть віртуальне оточення (рекомендовано):**
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. **Встановіть залежності:**
```bash
pip install -r requirements.txt
```

4. **Застосуйте міграції:**
```bash
python manage.py migrate
```

5. **Створіть суперкористувача:**
```bash
python manage.py createsuperuser
```

6. **Зберіть статичні файли:**
```bash
python manage.py collectstatic
```

7. **Запустіть сервер розробки:**
```bash
python manage.py runserver
```

8. **Відкрийте браузер та перейдіть на:**
```
http://127.0.0.1:8000
```

## 📖 Використання

### Реєстрація компанії

1. Перейдіть на сторінку реєстрації (`/accounts/register/`)
2. Оберіть "Компанія"
3. Заповніть форму:
   - Назва компанії
   - Податковий номер
   - Email та пароль
   - Вибір адреси на інтерактивній карті
   - Завантаження логотипу (опціонально)
4. Після реєстрації ви зможете створювати маршрути

### Реєстрація перевізника

1. Перейдіть на сторінку реєстрації (`/accounts/register/`)
2. Оберіть "Перевізник"
3. Заповніть форму:
   - Особисті дані
   - Тип та модель транспорту
   - Досвід роботи
   - Вибір адреси на карті
   - Опис послуг (опціонально)
4. Після реєстрації ви зможете переглядати маршрути та робити ставки

### Створення маршруту (Компанія)

1. Увійдіть як компанія
2. Натисніть "Створити маршрут" в навігації
3. Використайте карту для вибору точок відправлення та призначення:
   - Клікніть на карту для вибору точки відправлення
   - Клікніть на карту для вибору точки призначення
   - Або використайте пошук адреси
4. Заповніть інформацію про вантаж:
   - Тип вантажу
   - Вага (кг)
   - Об'єм (м³)
   - Ціна доставки
   - Дати забору та доставки
5. Натисніть "Створити маршрут"
6. Маршрут з'явиться у списку доступних для перевізників

### Зроблення ставки (Перевізник)

1. Увійдіть як перевізник
2. Перегляньте доступні маршрути через навігацію або на головній сторінці
3. Виберіть маршрут та натисніть "Зробити ставку"
4. У модальному вікні вкажіть:
   - Запропоновану ціну
   - Орієнтовну дату доставки
5. Натисніть "Залишити ставку"
6. Компанія побачить вашу ставку та зможе її прийняти

### Прийняття ставки (Компанія)

1. Перейдіть на сторінку деталей маршруту
2. Перегляньте список ставок від перевізників
3. Натисніть "Прийняти ставку" біля обраної пропозиції
4. Маршрут автоматично призначиться перевізнику

### Відстеження доставки

1. Перевізник оновлює прогрес доставки через сторінку відстеження
2. Компанія бачить поточну позицію вантажу на карті
3. Прогрес відображається у відсотках та на інтерактивній карті

### Оцінювання перевізників

1. Після завершення доставки компанія може зайти на профіль перевізника
2. Залишити оцінку (1-5 зірок) та коментар
3. Оцінки впливають на загальний рейтинг перевізника

### Чат

1. Відкрийте чат з перевізником/компанією через навігацію
2. Обмінюйтесь повідомленнями в реальному часі
3. Всі повідомлення зберігаються для історії

## 📊 База даних

Проєкт використовує такі основні моделі:

- **User** - користувачі системи (компанії та перевізники)
- **CompanyProfile** - профілі компаній (логіти, адреси, опис)
- **CarrierProfile** - профілі перевізників (транспорт, досвід, рейтинг)
- **Route** - маршрути доставки
- **Bid** - ставки від перевізників
- **Tracking** - відстеження прогресу доставки
- **Message** - повідомлення в чаті
- **Notification** - сповіщення користувачів
- **Rating** - оцінки перевізників

## 📝 Розробка
